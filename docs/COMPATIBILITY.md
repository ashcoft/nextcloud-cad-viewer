# Compatibility Requirements for Nextcloud CAD Viewer

## Nextcloud Version
The Nextcloud CAD Viewer is compatible with **Nextcloud 33**.

## PHP Requirements
The application requires **PHP 8.5** to function correctly. Ensure the PHP version meets this requirement to avoid any issues.

## Testing Procedures
To verify compatibility, follow these testing procedures:

1. **Setup**: Ensure you have Nextcloud 33 and PHP 8.5 installed in your development or testing environment.
2. **Installation**: Download and install the Nextcloud CAD Viewer app.
3. **Access Logs**: Monitor server logs for any errors or warnings during installation.
4. **Functionality Testing**: Perform basic functionality tests to ensure all features work as intended in Nextcloud 33 with PHP 8.5.
5. **Regression Testing**: If applicable, perform regression testing to ensure that previous functionalities remain operational.
6. **User Acceptance Testing**: Gather feedback from end-users to further verify compatibility and address any reported issues.

## Conclusion
Following these guidelines will help maintain compatibility for users of the Nextcloud CAD Viewer with the specified Nextcloud and PHP versions.
