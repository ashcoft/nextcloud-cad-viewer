# Changelog

All notable changes to this project will be documented in this file. The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-22
### Added
- Initial release of Nextcloud CAD Viewer app
- Support for DWG and DXF file viewing
- Full compatibility with Nextcloud 32 and 33
- Zoom, pan, and fit-to-view controls
- Fullscreen mode support
- Responsive design
- File content API endpoint
- Preview API endpoint
- Build system with webpack
- Development tooling (eslint, jest)
- GitHub Actions for automated building and releases
- Comprehensive documentation

### Features
- Web-based CAD file viewing without backend server
- Integration with mlightcad/cad-viewer library
- Easy update mechanism for upstream dependencies
- REST API for file operations
- Event-based architecture for extensibility
